Create Database in SSMS 7

CREATING A DATABASE IN SQL SERVER MANAGEMENT STUDIO, Follow the following steps:

  1. In your Start Menu go to your ‘SQL Server Management Studio’. Double Click on the ‘SQL Server Management Studio icon’.


       2.  It will take a while to open SQL Server Management Studio (SSMS). Wait for it to load.

   

3.  To connect to your server, choose your server name. Change Authentication mode to ‘Windows Authentication’. Click on ‘Connect’ to connect to your local server. 


 4.  Create your database in a new query window. New query window appears once you click on New Query. 


    5. Once new query window opens, create a Database. Syntax for creating a database is: CREATE DATABASE [SQL_HANDS_ON_SKILL_CHALLENGE]


6. After creating a database, Refresh and select your Database.


7. Drill down on the + sign at the right side of your Databases. Make sure it is visible on the object explorer.This Database created will be a Destination Database used to load tables from the source via SSIS.
