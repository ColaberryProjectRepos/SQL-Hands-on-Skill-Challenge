How to write Query efficiently to retrieve data from Database for business use?

Things to know before writing Query:

We must follow SQL QUERY ORDER OF EXECUTION which goes in the form of following sequence:

1: FROM <Defines Table as a source to query>

2: WHERE <Search Condition-Filters rows>

3: GROUP BY <Arranges rows by Groups>

4: HAVING <Search Condition-Filters Groups>

5: SELECT <Selects list - Which column to return>

6: ORDER BY <Sorts the output>
