Project Overview 1

Objective of this Project is to make you comfortable  in writing queries to retrieve and analyze data. 

There are three CSV files:


1. Accounts

2. Users

3. Revenue_Invoice_Info 

For convenience, these CSV were converted to Excel files.

These three files are  then loaded into the SQL server Database ( SQL HANDS ON TRAINING)  via ETL Process. 

Then the assigned questions were answered through querying process.
